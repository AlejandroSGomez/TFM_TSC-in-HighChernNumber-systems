# TFM_TSC-in-HighChernNumber-systems

## Requirements

This code was tested with:

- **Python** 3.12.2  
- **NumPy** 1.26.4  
- **SciPy** 1.13.1  
- **Matplotlib** 3.10.0  
- **Kwant** 1.5.0  
- **tqdm** 4.66.5  

You can install them all at once by running:

```bash
pip install numpy==1.26.4 scipy==1.13.1 matplotlib==3.10.0 kwant==1.5.0 tqdm==4.66.5
