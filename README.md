# TFM_TSC-in-HighChernNumber-systems

## Requirements

This code was tested with:

- **Python** 3.12.2  
- **NumPy** 1.26.4  
- **SciPy** 1.13.1  
- **Matplotlib** 3.10.0  
- **Kwant** 1.5.0  
- **tqdm** 4.66.5  

This repository’s scripts rely on two helper files, which are not included in standard pip packages:
	•	FuncsChern.py
	•	Modelos.py
